### 1. What is the primary purpose of a Content Delivery Network (CDN)?
A. To provide secure communication  
B. To reduce latency and improve content delivery  
C. To manage IP addressing  
D. To encrypt network traffic  

**Answer:** B  

---

### 2. A CDN mainly works by:
A. Increasing server processing speed  
B. Caching content closer to users  
C. Compressing all data packets  
D. Using larger bandwidth links  

**Answer:** B  

---

### 3. Which of the following is a key component of a CDN?
A. Edge servers  
B. Routers only  
C. Firewalls  
D. Load balancers only  

**Answer:** A  

---

### 4. CDN helps improve performance by reducing:
A. Packet loss  
B. Bandwidth  
C. Round Trip Time (RTT)  
D. Encryption overhead  

**Answer:** C  

---

### 5. Which type of content benefits MOST from a CDN?
A. Dynamic database queries  
B. Static content like images and videos  
C. Operating system files  
D. Internal server logs  

**Answer:** B  

---

### 6. Which routing technique is commonly used by CDNs to direct users?
A. Flooding  
B. Anycast routing  
C. Distance vector routing  
D. Link-state routing  

**Answer:** B  

---

### 7. A CDN reduces load on the origin server by:
A. Blocking user requests  
B. Replicating content across multiple servers  
C. Increasing server memory  
D. Reducing TCP connections  

**Answer:** B  

---

### 8. Which of the following is NOT a benefit of CDN?
A. Improved availability  
B. Reduced latency  
C. Increased server downtime  
D. Scalability  

**Answer:** C  

---

### 9. CDN is most commonly used by:
A. Operating systems  
B. Search algorithms  
C. Video streaming and web applications  
D. Network switches  

**Answer:** C  

---

### 10. Which metric is MOST improved by using a CDN?
A. CPU utilization  
B. Network latency  
C. Disk fragmentation  
D. Power consumption  

**Answer:** B
