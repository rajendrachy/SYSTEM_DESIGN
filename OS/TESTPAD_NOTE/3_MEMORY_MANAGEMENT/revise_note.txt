25. Calculate the size of memory if its address consists of 22 bits and the 
memory is 2-byte addressable.
Given:
Address bits = 22
Memory is 2-byte addressable

Number of addresses = 2^22
Total memory = 2^22 × 2 bytes
              = 2^23 bytes
              = 8 MB

Answer: Memory size = 8 MB







24. Given:
Page size = 2,048 bytes
Process size = 72,766 bytes
Memory is byte-addressable

Step 1: Find number of pages needed
Number of pages = Process size / Page size
                 = 72,766 / 2,048

2,048 × 35 = 71,680
72,766 − 71,680 = 1,086

So, process needs 35 full pages + 1 partial page = 36 pages total.

Step 2: Find internal fragmentation
Internal fragmentation = Page size − Last page data
                        = 2,048 − 1,086
                        = 962 bytes

Final Answer:
Internal Fragmentation = 962 bytes
